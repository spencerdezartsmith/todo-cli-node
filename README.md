# Command Line Today List

A small node todo-list that runs from the command line.

## Usage
- To add tasks:

```$ node task.js add "Buy milk"
Created task 1.

$ node task.js add "Buy eggs"
Created task 2.

$ node task.js add "Bake cake"
Created task 3.

$ node task.js add "Put groceries in the fridge"
Created task 4.```
